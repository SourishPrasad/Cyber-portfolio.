Termux Network Scanner Project

This project demonstrates a Python-based network scanner created inside Termux.

Tools Used

  • Termux
  • Python3
  • Socket library

Code Snippet

import socket

target = input("Enter target IP: ")
for port in range(1, 100):
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    socket.setdefaulttimeout(1)
    result = s.connect_ex((target, port))
    if result == 0:
        print(f"Port {port} is open")
    s.close()

Output

Shows which ports are open for a given target IP.
